Algorithms
==========

# Introduction

This includes exercises from the Algorithms lecture.

How to set-up:

1. Set GOPATH, e.g. ~/go
2. Copy algorithms repo to $GOPATH/src/algorithms

Now you are set. We are not using this as a Go module yet.

# Testing

For unit tests run:

``make test``

# Benchmarking

For running benchmars run:

``make bench``
